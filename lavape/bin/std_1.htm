This method is used to increment a state object of type Integer
by a certain amount (1. parameter) in order to avoid a
single-assignment violation.

Note that you cannot write for instance

   set n  <==  1;
   ...
   set n  <==  n+1;       // single-assignment violation!

in Lava, since this would be a single-assignment violation,
but you may write instead:

declare
   ~Integer n;              // state object of type Integer
do
   set n  <==  ~1;        // ~1 is an Integer 1 of category "state object"
   ...
   n.incBy(1)               // no single-assignment violation!
