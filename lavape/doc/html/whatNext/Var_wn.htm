Current selection: A reference to a Lava variable.

Your next edit step may be one of the following:

  • Select a member variable from the "Members " combo-box at the top of this window (if a member of the required type exists!).
  • Press the "Delete" key or click the "Cut" button to delete the variable reference from the selected member onward to the right. If you had selected the first (leftmost) or the only variable reference in a "dotted" sequence of member references then the entire variable reference will be replaced by a placeholder <var> or <expr>, depending on whether the variable reference had been produced by replacing a <var> or an <expr> originally. (<var> stands for the target of an assignment, i.e., a variable in "write position", <expr> for an executable Lava expression, which may also be a variable in "read position").

    Example: If you select and delete the "c" in a.b.c.d, then a.b will remain. If you select and delete the "a", then <var> or <expr> will remain.
  • Click the "Go to declaration" button to jump to the declaration of the selected variable, for instance to the declaration of "b" in a.b.c.d if "b" is selected.
  • Click the "Find references" button to list all or particular (specifiable) references to the selected variable.
