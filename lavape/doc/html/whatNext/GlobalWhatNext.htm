Currently there is no document open in LavaPE. In this state you can primarily use

  • File menu ==> New Lava file to create a new and empty Lava file, or
  • File menu ==> Open to open an existing Lava file, or
  • File menu ==> New component, to create a new component object implementation, or
  • File menu ==> Exit to terminate LavaPE, or
  • you can select and open a file from the file history at the bottom end of the file menu.

(You may also use the "New Lava file ", "New component", or "Open file" buttons on the file toolbar, of course.)
