File menu commands

The File menu offers the following commands:

New Lava file Creates a new Lava file.
New component Creates a new Lava component file.
Open Opens an existing document.
Close Closes an opened document.
Save Saves an opened document using the same file name.
Save As Saves an opened document to a specified file name.
Include Include another Lava file.
Generate linked HTML files Generates linked HTML files from the current Lava file.
Generate single HTML file Generates a single HTML filewithout links from the current Lava file.
Exit Exits Lava.
