View menu commands

The View menu offers the following commands:

File toolbar Shows or hides the file toolbar.
Tree oper. toolbar Shows or hides the tree operation toolbar.
Declaration toolbar Shows or hides the declaration toolbar.
Details toolbar Shows or hides the details toolbar.
Oper. symbol toolbar Shows or hides the operation symbol toolbar.
Keyword toolbar Shows or hides the keyword toolbar.
References toolbar Shows or hides the references toolbar.
Status Bar Shows or hides the status bar.
