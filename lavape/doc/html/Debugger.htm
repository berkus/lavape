The interactive Lava run time debugger

LavaPE provides an interactive run time debugger, which allows you to set breakpoints, to step from statement to statement and into functions, and to inspect the values of local variables and formal parameters.

The debug tool bar

provides the following operations (from left to right):

  • Start/continue debugging.

  • Stop at next statement.

  • Stop at next function call.

  • Step into function.

  • Execute and then leave the current function.

  • Continue up to the selected construct.

  • Set/remove breakpoint at the selected construct.

  • Remove all breakpoints.

  • Stop/abort the current debugging session.
