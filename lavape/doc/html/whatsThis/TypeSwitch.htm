Type switch statement

The Lava type of statement is similar to the Lava switch statement.

Differences:

  • It branches over the run time type of the result object of its switch expression.
  • Every case clause specifies a single (real or virtual) type and introduces a new local variable of that type whose scope is the respective branch. The result object of the switch expression is assigned to this variable.

The else branch is optional and may be deleted using the DEL key or the scissors button.

W.r.t. variable initialization checks, the same remarks apply as for the if statement.

Don't use the type switch statement unless really unavoidable!
