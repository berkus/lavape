Select

Use the select expression

select 
  [type]<varName> from <set>
where
  <stm>
add
  <expr>
to
  <expr>
#select
to search a finite set of Lava objects for objects that fulfill a certain condition (expressed by the optional [type] specification and the where clause), evaluate the first expression and add its result object to the set that is represented by the value of the second expression.
