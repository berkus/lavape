Scale expression

Use the scale expression to "raise" the class of its input object to a more derived class, e.g., the class "double" of the number 3.5 to the class meters (derived from class "double", say), yielding 3.5 meters.

This is accomplished essentially by an implicit "copy onto" operation which you could also perform explicitly by using the copy statement.

Using explicit scale units rather than dimensionless numbers provides more type safety and might prevent costly failures of Mars missions caused by faulty computer programs that confuse yards with meters.

See the ScaleUnits.lava sample.
