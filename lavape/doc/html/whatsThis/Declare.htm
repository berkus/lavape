Declare

Use declare to declare local variables which can be referenced only within the body of the declare construct.

LavaPE  reports an error if you declare a local variable but do not use it.

A local variable must be initialized before using it in read position, i.e., a value must be assigned to it. This value may either be a constant (Boolean, string, numeric constant, etc.), or a new Lava object that results from the execution of a new expression.
