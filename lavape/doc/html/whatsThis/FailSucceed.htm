Affirm/negate statements

Use the affirm statement to induce an immediate affirmative/successful return from the current exec. If the function or operator has been invoked as a statement then the resulting truth value of that statement is "true".

Note: If you are in a function with mandatory output parameters then affirm is admissible only at places in the function body where values have already been assigned to all mandatory output parameters. (This is checked and enforced by LavaPE!)

Use the negate statement to induce an immediate negative/unsuccessful return from the current exec. If the function or operator has been invoked as a statement then the resulting truth value of that statement is "false".

negate has an optional exception parameter, i.e., an expression whose value will be thrown as an exception object if present.

Note: If negate is used at a place in a function body where not yet all mandatory output parameters have been assigned a value then you must throw an exception in order to avoid inadvertent use of undefined outputs by the caller. (This is checked and enforced by LavaPE!)

See also the assert statement.
