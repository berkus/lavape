Switch statement

The Lava switch statement largely corresponds to the switch statement in C and C++.

Differences:

  • It doesn't provide a "break" statement to terminate a branch, and therefore it isn't possible to run from one branch into the next branch as in C/C++ if the "break" is missing.
  • If multiple cases are handled by the same branch, then you do not provide multiple case clauses in Lava switch but you provide multiple alternative expressions in one case clause.

The else branch is optional and may be deleted using the DEL key or the scissors button.

W.r.t. variable initialization checks, the same remarks apply as for the if statement.
