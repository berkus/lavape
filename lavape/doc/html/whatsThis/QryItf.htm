"Query interface" for a component object

First use the attach expression to make an existing component object accessible through one of its component object interfaces.

For example

declare
  DOM itf1;
  I2 itf2
do
  set itf1 <== attach HTTP,
               interface DOM,
               url "http://lavape.sourceforge.net";
  ...

  set itf2 <== interface I2 of itf1
#declare

could be used to access the Lava home page through a "Document Object Model" interface (not yet implemented).

Thereafter you can use the "query interface" expression "interface <type> of <expr>" to make another interface (of type I2) accessible that is also supported by this component object.
