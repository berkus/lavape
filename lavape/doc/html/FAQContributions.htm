How can I contribute to the Lava development?

The development of a completely new programming language and of an entire programming environment is a very ambitious and comprehensive goal that cannot be accomplished by two people in a reasonable amount of time. So contributions of all kinds are not only highly welcome but simply indispensable.

Lava is an Open Source project. You can get the most recent Lava sources through the CVS service of the BerliOS Open Source Mediator. (CVS stands for "Concurrent Versions System".) For more details see the green box near the top of the Lava Home Page.

  • You can either send us modified versions of Lava sources and leave it to us to merge them with the current version in the BerliOS repository,
  • or, if you become more engaged and if you cooperate with the Lava developer team on a more regular basis, we can admit you to the group of "official" Lava developers. Then you can directly check in your modified sources to CVS.

See here for contributions that are particularly desirable from our present point of view.
