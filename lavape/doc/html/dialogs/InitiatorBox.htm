Initiator property sheet

Name

The name of an initiator may contain only letters, digits, and "_". The first character must not be a digit.

Transaction

An initiator may be executed as an atomic transaction.

Synchronous

If an initiator is executed in synchronous mode then the caller waits until it returns.

Concurrent

If an initiator is executed in concurrent mode then a concurrently executing thread is allocated and assigned to it.

Note: Initiators cannot throw exceptions and never have output parameters.
