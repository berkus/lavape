Set property sheet

Name

The name of a set may contain only letters, digits, and "_". The first character must not be a digit.

Lava sets are multi-sets: the same object may occur more than once as an element in a given set. Member function Contains counts the number of occurrences of a given object in a given set. The "<expr> in <set>" statement checks whether an object is contained (at least once) in a set.

Select set element type

The type of the set elements may be selected from the combo-boxes "Basic interfaces" and "Other interfaces". "Basic interfaces" lists all interfaces whose declaration is contained in the basic include file std.lava.
