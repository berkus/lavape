Package property sheet

Name

The name of a package may contain only letters, digits, and "_". The first character must not be a digit.

Opaque

You can declare a package to be opaque to prevent access to the contained declarations from the "outside world". In this way you can achieve similar effects as with the private/friend declarations in C++ or with inner classes in Java.

Select packages to be extended

The Lava pattern notion is based on multiple inheritance for packages with pattern parameters ("virtual types"). Here you can specify which packages shall be extended by the current package.
