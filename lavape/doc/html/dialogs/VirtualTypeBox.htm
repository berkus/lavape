Virtual type property sheet

Name

The name of a virtual type (or "pattern parameter") may contain only letters, digits, and "_". The first character must not be a digit.

Enable renaming

You may change the names of functions, member variables and virtual types when you override them in a derived interface or pattern (for instance in order to resolve name collisions in cases of multiple inheritance). They inherit the names of the base versions by default (and their name is changed accordingly if the inherited base name is changed). Check this check-box if you would like to assign a different name (which will not be changed then if the base name is changed).

Select value type of VT

If a virtual type is not abstract then you must specify an existing type as its value. This may be a concrete type or may again be virtual and may be selected from the combo-boxes "Basic interfaces" and "Other interfaces". "Basic interfaces" lists all interfaces whose declaration is contained in the basic include file std.lava.

Remove overrides

Transforms an overridden virtual type into a non-overridden virtual type, as if it had been newly declared in the containing pattern.

Abstract

A virtual type (pattern parameter) of a package or of a non-creatable interface may be abstract. I.e., the specification of a value type for this pattern parameter is deferred to a derived pattern. For an abstract virtual type you may also defer the decision upon the object category until it is overridden by a non-abstract virtual type in a derived pattern. See next paragraph.

Defines object category

A virtual type may or may not determine the object category of all objects declared with this VT.

Object category

If you have decided that the virtual type shall determine the object category of all objects declared with this VT then you must select this category here.

Kind of element reference

If this VT defines the type of the elements of a set, chain, or array then you may specify whether the elements are treated as constituents, acquaintances or reverse links.
