Release notes

LavaPE programming environment

The Lava programming environment LavaPE is being developed under Windows XP, but should run also under all earlier 32-Bit Windows systems. (On Win2000/XP systems you will need administrator privileges for the installation as usual.)

Please note: LavaPE software + online help and particularly the Lava interpreter are still under development and incomplete. If you wish to get an early impression of the new look and feel of Lava programming by playing around with the current LavaPE preview version: Just download the self-extracting installation file

  • LavaPEpreview-0.6.4.exe   Rel. 0.6.4, ~1.6 MB, released 2002-12-10, updated 2003-01-06.
  • LavaPEpreview-0.6.3.exe   Rel. 0.6.3, ~1.6 MB, released 2002-09-17, updated 2002-11-21.

and execute it. LavaPE will be installed to your "<program files>\LavaPE" directory if you don't specify a different installation directory during the setup process. A LavaPE folder is created on your desktop. Click the volcano icon in this folder to launch LavaPE. (In case of crashes please send us the respective *.lava file(s) together with instructions how to produce the crash.)

Some users have informed us that LavaPE crashed immediately when started: they did not yet have the current version of Mfc42.dll. (You have it if you have Internet Explorer 4 or 5.) In this case you can download Mfc42.zip and unpack Mfc42.dll into the LavaPE installation directory. Thereafter LavaPE should come up without crash.

You can uninstall LavaPE by double-clicking the "Uninstall LavaPE" icon in the LavaPE desktop folder or in the usual way via Control Panel / Software / Remove. You should terminate Lava/LavaPE before installing a new version, but you may install it over an existing installation.

The download comprises the LavaPE online help file LavaPE.chm (see below) because the latter is needed for the context-sensitive online help of LavaPE. So you need not download it separately. But in any case you must make sure that the MS help viewer "hh.exe" (see below) has been installed or else you won't be able to use the LavaPE online help.

Lava online documentation

A) On Windows NT/9X/2000/XP platforms you may download a separate help file

  • LavaPE.chm (~680 KB)

if you don't want to install LavaPE (which contains this version of the online help). Simply double-click "LavaPE.chm" to open it.

This version has the advantage that it comprises full-text search, print support and seems to function more reliably than the more portable HTML version offered below. Note that the Lava sample programs in the section "Learning Lava by samples" of the Lava online documentation can be launched only from LavaPE.chm and only if you have installed LavaPE. The (automatically produced) HTML transcription of the samples can be viewed in any case.

Make sure that the (free) MS help viewer "hh.exe" is in your Windows or Winnt directory: You can get the self-extracting installation file "hhupd.exe" from Microsoft or from us. Users of Windows 2000 and XP should not download and execute hhupd.exe since these systems have their own built-in version of hh.exe.

B) The portable HTML version should be viewable with all browsers that support frames, style sheets, Java, JavaScript:

LavaDoc.zip (~1 MB)

LavaDoc.tgz (~750 KB)

Unpack to a proper directory of its own and double-click "LavaWebDoc.htm".

Note: The table of contents is a Java applet in case B; so you should enable Java.
